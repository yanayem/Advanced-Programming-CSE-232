# Advanced Programming – CSE 232

This repository contains Java example files and assignments for the **Advanced Programming (CSE 232)** course.

## 📁 Project Structure

- `src/` – Java source code files
- `examples/` – Example programs
- `assignments/` – Assignment solutions


## 📚 Detailed Topics Covered

### ✅ 1. Programming Fundamentals
- Variables, data types, operators  
- Input/output operations  
- Type casting  
- Memory concepts  

### ✅ 2. Control Structures
- Conditional statements (if, else-if, switch)  
- Looping constructs (for, while, do‑while)  
- Nested loops  
- Flow control & debugging 

### ✅ 3. Functions & Modular Programming
- Function declaration & definition  
- Parameter passing (value/reference)  
- Recursion  
- Scope & lifetime of variables  

### ✅ 4. Arrays & Strings
- 1D & 2D arrays  
- Searching & sorting  
- String manipulation  
- Character arrays vs string libraries  

### ✅ 5. Object‑Oriented Programming (OOP)
- Classes & objects  
- Constructors & destructors  
- Encapsulation  
- Inheritance  
- Polymorphism  
- Abstraction 

### ✅ 6. File Handling
- Reading/writing files  
- Working with text & binary files  
- Error handling  

### ✅ 7. Algorithms & Problem Solving
- Time complexity basics  
- Searching algorithms  
- Sorting algorithms  
- Real‑world problem breakdown
